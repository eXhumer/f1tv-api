# f1tv-api

Unofficial F1TV API client for JavaScript/TypeScript.

## Installation
To install this package from npm registry:
  ```bash
  npm install @exhumer/f1tv
  ```

To install this package from Git:
  ```bash
  npm install git+https://github.com/eXhumer/f1tv.git
  ```
